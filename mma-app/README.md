# mma-app

React Native app scaffolded for Expo with TypeScript, Expo Router navigation, ESLint, and Prettier.

## Run locally (Windows)

From PowerShell or Command Prompt:

```powershell
cd mma-app
npm install
npm run start
```

Then press:

- `a` for Android emulator/device
- `i` for iOS simulator (macOS only)
- `w` for web

Or run directly:

```powershell
npm run android
npm run ios
npm run web
```

## Quality tools

```powershell
npm run lint
npm run lint:fix
npm run format:check
npm run format
```
